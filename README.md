# Ice Cream OOP Assignment

This project shows Object-Oriented Programming (OOP) using an Ice Cream dessert system. The program uses classes and inheritance to organize different ice cream flavors.

## Table of Contents
- About
- Features
- Installation

## About

This project was made for an OOP assignment where we needed to create a hierarchy of related objects.

The parent class is **Dessert**, which represents the general dessert category.

The child class is **IceCream**, which extends **Dessert** and represents ice cream desserts.

The grandchild classes extend **IceCream** and represent different ice cream flavors:

- ChocolateIceCream  
- VanillaIceCream  
- StrawberryIceCream  
- CookiesAndCreamIceCream  


## Features

- Uses OOP class hierarchy
- Uses properties to describe objects
- Uses methods for actions
- Uses a trait to share functionality
- Uses `spl_autoload_register` to load classes


## Installation

1. Clone the repository  
2. Open the project folder  
3. Run the project using a local PHP server like MAMP

The **index.php** file loads the classes and runs the program.
